# Credit Card Fraud Detection using Machine Learning

This project builds a machine learning model to detect fraudulent credit card transactions.

Dataset: Kaggle Credit Card Fraud Dataset

Technologies:
- Python
- Pandas
- Scikit-learn

Steps:
1. Data preprocessing
2. Handling class imbalance using undersampling
3. Logistic Regression model training
4. Model evaluation using confusion matrix and classification report

Results:
Accuracy: 94%
